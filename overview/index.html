    Online Account Opening API Documentation

    Using the API

    This API is based on REST Principle which lets you create a new account with Sterling Bank. By default, all request to https://pass.sterling.ng/CAMS receives the v1 version of the REST API.

     

    The API access is over HTTPS and it is accessed from https://pass.sterling.ng/CAMS. All data is sent and received as JSON. The endpoints return a JSON Metadata which gives you the details of a user, provided by BVN. The API provides a set of endpoints, each with its path for creating an account.

    Request

    The Online Account Opening APIs are accessed through a standard HTTPS request in UTF-8. The following methods are available for this API.

    METHOD ACTION
    POST CREATE RESOURCES

    Parameter

    Parameter Type Description
    id string user ID
    bvn string Bank Verification Number
    dateOfBirth date Date of Birth
    email string Email Address
    phoneNumber string Users Mobile Number
    address string Residential Address
    country string Country of Origin
    state string Country State
    progressLevel string Porgress level status
    isActive Boolean Active status of the user (true,false)
    firstName string User first name
    middleName string User middle name
    lastName string User last name
    sector string Users Occupation
    industry string The company or industry where the user works
    nationality string Nationality e.g Nigerian,Ghanian,American...
    gender string Gender e.g Male,female...
    title string Your name Title eg Mr,Mrs,Miss
    interestedInLoanProducts string shows if user is interested in loan products or not
    maritalStatus string Marital status e.g single,married...
    numberOfChildren string Number of Children the user have
    occupation string User Occupation
    hobby string User Hobby
    nokFirstName string The first name of the Next Of Kin
    nokMiddleName string The Middle name of the Next Of Kin
    nokFullName string The full name of the Next Of Kin
    nokLastName string The last name of the Next Of Kin
    nokAddress string Address of the Next Of Kin
    nokPhoneNumber string Mobile Number of the Next Of Kin
    nokRelationship string The relationship between the Next Of Kin and the user e.g Mother,Father,Aunt...
    employmentStatus string Employment status of the user e.g self employed,retired,student..
    employerName string Name of Employer/Company
    employmentAddress string Address or employment
    addressTown string Town where employment is located
    addressState string State where employment is located
    addressLandmark string Landmark where employment is located e.g kfc
    accountDetails object This hows the account details of the user, details of the user can be view below this table
    addedDate date-time Date of added Information example - 2019-11-20T13:26:22.326Z
    modifiedDate date-time Date of modified Information example - 2019-11-20T13:26:22.326Z
    sessionKey string The Session key of every transaction
    passportPhoto string user passport image directory
    validIdType string Type of Identification eg: Drivers license
    validIdNumber string Valid Identification Number
    signature string User signature
    validId string Image directory of Valid Identifiaction
    reference string the reference form file
    bvN_Firstname string First name of Bank Verification Number
    bvN_Lastname string Last name of Bank Verification Number
    isDefaultPassword true Default Password status
    responseCode string The response result Code
    responseDescription string The response result message
