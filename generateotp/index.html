    Generate OTP

    POST /api/User/GenerateOTP
    This generate a new OTP, sending it to email and mobile number.

    Example

    {
  "mobile": "string",
  "email": "string"
}

     

    Parameters

    Parameter Type Description
    mobile string *required
    Mobile number to send OTP
    email string *required
    Eamil address to send the OTP

    Responses

    result

    {
  "responseCode": "string"
}
    200 Schema Type Description
    responseCode string The response result Code
