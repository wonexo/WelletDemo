    ONEXPRESS

    Corporate Account Details TO 1Xpress

    POST /api/User/CorporateAccountDetailsToOneXpress
    This API method sends the details of the corporate account created to 1Xpress

     

    request

    {
    "dataFields": {
        "AccountNumber": "string",
        "BranchCode": "string",
        "BranchName": "string",
        "BVN": "string",
        "CustomerID": "string",
        "DateOfIncorp": "string",
        "Industry": "string",
        "InternalRef": "string",
        "ProductName": "string",
        "RCNumber": "string",
        "RefereeAccountNumber1": "string",
        "RefereeAccountNumber2": "string",
        "RefereeBank1": "string",
        "RefereeBank2": "string",
        "Sector": "string",
        "Source": "string",
        "TIN": "string"
    }
}
    Parameter Type Description
    AccountNumber string Users Account Number
    BranchCode string Account Branch Code used in Openining the Account
    BranchName string Name of branch
    BVN string Bank verification Number
    CustomerID string Customers Identifiaction Number
    DateOfIncorp string Date of incorporation
    Industry string The company or industry where the user works
    InternalRef string
    ProductName string Name of Product
    RCNumber string Company registration Number
    RefereeAccountNumber1 string First Referee Account Number
    RefereeAccountNumber2 string Second Referee Account Number
    RefereeBank1 string First Referee Bank
    RefereeBank2 string Third Referee Bank
    Sector string Users Occupation
    Source string source of the request e.g one bank , specta
    TIN string Tax Identification Number

    This API method sends documents such as CAC registration, Tax etc. about a company to 1Xpress

    {
    "accountNumber": "string",
    "documentName": "string",
    "docURL": "string"
}
    Parameter Type Description
    accountNumber string Users Account Number
    documentName string Name of Document
    docURL boolean Document Url

     

    1Express Singatory Table Request

    POST /api/User/OneXpressSignatoryTableRequest
    This API method sends information about a Director or Signatory to 1Xpress.

     

    request

    {
    "accountNumber": "string",
    "surname": "string",
    "otherName": "string",
    "fIrstName": "string",
    "bvn": "string",
    "phone": "string",
    "iSsignatory": "string",
    "mandateAuth": "string",
    "classOfSig": "string",
    "mandateInstruction": "string",
    "identificationType": "string",
    "idNo": "string",
    "signature": "string",
    "passportPic": "string",
    "mandate": "string",
    "urlPhoto": "string",
    "urlSignature": "string",
    "urlMandate": "string",
    "idName": "string"
}
    Parameter Type Description
    AccountNumber string Account Number
    surname string Surname or last name
    otherName string Other name
    fIrstName string First name
    bvn string Bank verification Number
    phone string Mobile Number
    iSsignatory string
    mandateAuth string MAndate Authorisation
    classOfSig string Signature Class
    mandateInstruction string Mandate Instruction
    identificationType string Identification type
    idNo string Identification Number
    signature string Signature
    passportPic string Passport photograph
    mandate string Mandate name
    urlPhoto string URL of photograph
    urlSignature string URL of signature
    urlMandate string URL of mandate
    idName string Identification Name
