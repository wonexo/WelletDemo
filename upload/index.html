    Upload Document To T24

    POST /api/User/UploadDocumentToT24
    This method passes the following users documents to T24.

    Example

    Here are the object to pass to T24. 

      {
    "document": {
      "compcode": "string",
      "customerid": "string",
      "signatoryClass": "string",
      "mandateDesc": "string",
      "signatoryShortName": "string",
      "signatoryName": "string",
      "filename": "string"

    }
  }

    Parameters

    Parameter Type Description
    document object *required
    The Document contains list of objects to be passed below and their description
    compcode string *required
    T24 Company Code
    customerid string *required
    T24 Customer Id
    signatoryClass string *required
    Class of Signatory
    mandateDesc string *required
    Description of Mandate
    signatoryShortName string *required
    Short name Of Customer
    signatoryName string *required
    Full Customer Name
    filename string required
    Upload File Name (has to be of file format     .jpg*)

    Responses

    result (success)

    {
  "responseCode": "1",
  "responseId": "string",
  "responseText": "string"
}

    result (error)

    {
  "responseCode": "1",
  "responseId": "string",
  "responseText": "IMAGE.REFERENCE:1:1=MISSING CUSTOMER - RECORD, IMAGE.REFERENCE:1:1=INPUT MISSING"
}
    Parameter Type Description
    responseCode string The response result Code
    responseId string The response result Id
    responseText string The response result Description eg Success

    This method passes the following users upload image to Azure.

    "fileName": "string",

  }

    Parameters

    Parameter Type Description
    appId string *required
    Application Identification Number
    folderName string *required
    Name of folder
    fileName string *required
    Name of File
    base64String string *required
    base64 Encrypted String

    Upload Document To CAMU

    POST /api/User/UploadDocumentToT24
    This method passes the following users documents to T24.

    Example

    Here are the object to pass to T24. 

      {
    "folio": "string",
    "dataFields": {
      "accountName": "string",
      "accountNumber": "string",
      "branchCode": "string",
      "bvn": "string",
      "source": "string",
      "disContactPersonAdd": "string",
      "disContactPersonMobile": "string",
      "disContactPersonName": "string",
      "dob": "string",
      "emailAddress": "string",
      "firstName": "string",
      "lastName": "string",
      "middleName": "string",
      "mobileNumber": "string",
      "residentialAddress": "string",
      "urlMeansOfID": "string",
      "urlOther": "string",
      "urlPhotoID": "string",
      "addState": "string",
      "urlReference1": "string",
      "urlReference2": "string",
      "urlMandate": "string"
    }
  }

    Parameters

    Parameter Type Description
    folio object portfolio group where the request belong to on 1Xpress. eg one bank, specta, etc
    accountName string Name of users account
    accountNumber string Account number
    bvn string Bank Verfication Number
    source string source of the request e.g one bank , specta
    disContactPersonAdd string Add disContact person
    disContactPersonMobile string disContact Person Mobile number
    disContactPersonName string DisContact person name
    dob string Date Of Birth
    emailAddress string Email Address
    firstName string Account First Name
    lastName string Account Last Name
    middleName string Account Middle Name
    mobileNumber string Account Mobile number
    residentialAddress string Users ressidential address
    urlMeansOfID string link to means of Idenfication
    urlOther string Other available link
    urlPhotoID string Link to Photo identification
    addState string add residential address
    urlReference1 string First Account Reference
    urlReference2 string Second Account Reference
    urlMandate string A picture containing a signature and image of the customer
